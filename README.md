## Web destinada a la búsqueda de de gifs online.
#### Esto es una práctica para aprender el uso de React + TypeScript utilizando CustomHooks, Componentes aislados, useState, useEffect, API fetch, etc.

# Link para visitar el sitio Web
https://gifapp-prueba.netlify.app/

#### Si desea descargar el repositiorio, debe seguir los siguientes comandos en la carpeta del directorio, para instalar las dependencias:

#### Para instalar todas las dependencias
- npm install

#### Para ejecutar en modo desarrollo
- npm run dev
